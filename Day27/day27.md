# Day 27 – Prior Authorization Story Simulator

## 📌 Objective
Built an interactive **Prior Authorization Story Simulator** using a single HTML file to understand healthcare insurance workflows through conversational storytelling.

---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- createElement() + appendChild() DOM manipulation
- Responsive UI

---

## 📖 Project Overview

This simulator explains the Prior Authorization (PA) process using an interactive story between:

- 👦 Rahul (Patient)
- 👧 Priya (Healthcare Operations Specialist)

The application walks users through the complete healthcare authorization journey in **8 interactive scenes**, making a complex insurance workflow easy to understand.

---

## 🎯 Features

- Interactive chat-based storytelling
- Progress bar for all 8 scenes
- Branching choices after every stage
- Educational healthcare workflow
- Clean medical-themed UI
- Append-only chat feed
- No `innerHTML` used for chat rendering
- Fully responsive design

---

## 🧩 Story Flow

1. Doctor Visit
2. Insurance Roadblock
3. Understanding Prior Authorization
4. Insurance Review Process
5. PA Denial
6. Appeal Submission
7. Approval
8. Final Takeaways
