# Day 37 – Task Compass

## 📌 Objective

Built an interactive **Task Compass** simulation using Claude AI to understand how work flows through real organizations through ownership, delegation, workflow routing, and collaboration.

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

---


---

## ✨ Features

- Interactive Tech Company workplace simulation
- Three gameplay stages
- Drag-and-drop role assignment
- Task workflow routing
- Collaboration challenge
- Ownership reasoning and supporting roles
- Organizational Thinking Dashboard
- Progress tracking
- Animated transitions
- Glassmorphism dark UI
- Responsive offline application
- Replay with randomized workplace scenarios

---

## 📚 Key Learnings

- Every workplace task has a primary owner, but successful delivery often requires collaboration.
- Understanding task routing improves workflow efficiency.
- Delegation helps distribute responsibilities effectively.
- Organizational problems are solved through coordinated teamwork rather than individuals.
- Clear ownership reduces confusion and improves accountability.
- Interactive simulations make organizational concepts easier to understand.

---

## 🚀 Outcome

Successfully generated and tested a complete browser-based **Task Compass** application using Claude AI.

The project demonstrates organizational workflow, ownership, delegation, collaboration, and management thinking through an engaging interactive simulation built entirely with HTML, CSS, and Vanilla JavaScript.
