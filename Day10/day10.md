# Day 10 – Build Your Personal Portfolio Website

## Project Overview
Today I built my first personal portfolio website using AI.

The portfolio showcases my profile, projects, achievements, skills, and contact information in a modern and professional format. The goal was to create a strong personal brand and an online presence that can be shared with recruiters and professionals.

---

## Features Implemented

- Responsive Personal Portfolio Website
- Modern Dark Theme UI
- Hero Section with Introduction
- About Me Section
- Skills Showcase
- Projects Section
- Achievements & Milestones Timeline
- Contact Section
- GitHub, LinkedIn & Email Links
- Mobile-Friendly Design
- Smooth Navigation

---

## Technologies Used

- HTML
- Tailwind CSS
- JavaScript
- Claude AI

## Key Learnings

- Learned the importance of personal branding.
- Understood how AI can generate complete websites from structured prompts.
- Improved website design and layout understanding.
- Learned how to showcase projects professionally.
- Experienced AI-assisted web development workflow.
