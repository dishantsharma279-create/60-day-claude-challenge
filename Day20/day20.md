# Day 20 – AI Face Puzzle Game

## Objective
Built an AI-powered Face Puzzle Game using Claude AI as a single self-contained HTML application.

## Features
- Webcam access using getUserMedia()
- Live camera preview
- Capture photo
- Puzzle generation (3×3, 4×4, 5×5)
- Drag & Drop support
- Touch support for mobile
- Timer and Move Counter
- Correct Piece Counter
- Win Detection
- Results Screen
- LocalStorage Leaderboard
- Play Again
- Retake Photo
- New Photo
- Responsive UI

## What I Learned
- Browser Camera API
- Image processing using HTML Canvas
- Puzzle generation logic
- Drag & Drop interactions
- LocalStorage implementation
- Building complete browser applications with AI
