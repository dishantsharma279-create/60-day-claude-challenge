# Day 36 – Cognitive Pattern Explorer

## Overview
Today I built **Cognitive Pattern Explorer**, a psychology-inspired interactive web application that encourages self-reflection through engaging scenarios and decision-making activities.

The application is built as a **single self-contained HTML file** using **HTML, CSS, and Vanilla JavaScript**. It works completely offline without any external libraries or APIs.

---

## Features

- Calm Mode & Focus Mode
- Interactive Thinking Style Discovery
- Drag-and-Drop Priority Ranking
- Thinking Timeline Mapping
- Personalized Reflection Journal
- Thinking Pattern Breakdown
- Progress Tracking
- Responsive UI
- Modern Animated Interface
- Fully Offline

---

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

---



## Key Learnings

- Learned how educational experiences can be built without making clinical assessments.
- Improved understanding of interactive UI design using Vanilla JavaScript.
- Practiced drag-and-drop interactions and reusable components.
- Built a responsive single-file application.
- Learned how personalized feedback improves user engagement.
