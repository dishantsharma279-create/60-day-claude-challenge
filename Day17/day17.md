# Day 17 – AI Vehicle Cost & Fuel Analysis Dashboard

## Objective
Built an AI-powered Vehicle Cost & Fuel Analysis Dashboard using Claude AI and a CSV dataset.

## Features
- Fuel Cost per KM Analysis
- CO₂ Emission Comparison
- Maintenance Cost Analysis
- Age-wise Cost Comparison
- E85 Break-even Analysis
- Interactive SVG Charts
- KPI Cards
- Fuel Comparison Dashboard

## Key Insights
- EV had the lowest running cost per km.
- Petrol offered faster refueling but higher monthly fuel costs.
- E85 fuel was cheaper at the pump but resulted in a slightly higher running cost.
- Dashboard made cost, maintenance, and environmental impact easy to compare.

## Learnings
- Learned how Claude can analyze CSV datasets.
- Generated a complete HTML dashboard without writing code.
- Understood KPI calculations and SVG-based data visualization.
- Improved prompt engineering for analytics workflows.
