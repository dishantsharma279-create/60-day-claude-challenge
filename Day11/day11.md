# Day 11 — ATS Resume Optimizer & Resume Generator

## Project Overview
Today I learned how Applicant Tracking Systems (ATS) evaluate resumes and how AI can help optimize resumes for better recruiter visibility.

## ATS Optimization Process
- Analyzed my existing resume against a target AI Software Development Intern job description.
- Identified missing keywords, skills, and improvement areas.
- Improved resume structure and professional wording while maintaining complete factual accuracy.

## Key ATS Improvements
- Added better role-relevant keywords related to AI tools, prompt engineering, Git, GitHub, automation workflows, and software development.
- Improved professional summary and skills organization.
- Enhanced project descriptions to better showcase practical AI learning and GitHub documentation.

## Key Learnings
- ATS systems prioritize relevant keywords and clear formatting.
- A strong resume should match job requirements without adding false information.
- AI can help improve resume readability and professional presentation.
- Continuous projects and skill-building improve career readiness.

## Tools Used
- Claude AI
- ATS Resume Optimization Framework
- Git & GitHub

## Conclusion
This project taught me how AI can transform a basic resume into a more ATS-friendly and recruiter-ready document while keeping all information accurate and honest.
