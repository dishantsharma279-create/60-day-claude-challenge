# Day 33 – Media Integrity Analyzer

## Project Overview
Today I built **Media Integrity Analyzer**, a single-file HTML application that teaches media literacy through interactive learning instead of traditional quizzes.

The application helps users understand how misleading headlines, emotional manipulation, and audience targeting influence the way information is consumed.

---

## Features

- Single HTML file (Offline Application)
- Pure HTML, CSS & Vanilla JavaScript
- Responsive Editorial Dark UI
- Multiple Color Themes
- Interactive Headline Detective
- Emotion Detector Challenge
- Live Media Integrity Metrics
- Headline Accuracy Analysis
- Emotional Manipulation Analysis
- Source Reliability Tracking
- Audience Targeting Insights
- Media Integrity Dashboard
- Replay with Randomized Scenarios

---

## Workflow

1. Choose a preferred color theme.
2. Start the guided media literacy lesson.
3. Complete the Headline Detective challenge.
4. Identify exaggerated and misleading phrases.
5. Review the rewritten balanced headline.
6. Analyze emotional manipulation techniques.
7. Observe live integrity metrics.
8. Review the final Media Integrity Dashboard.
9. Replay with completely new scenarios.

---



---

## Key Learnings

- Headlines often exaggerate facts to increase clicks.
- Emotional wording influences reader perception.
- Reliable information requires critical thinking instead of instant reactions.
- Neutral language usually represents information more accurately.
- Audience targeting and emotional framing are common techniques used in digital media.
- Interactive educational applications make complex concepts easier to understand.
