# Day 31 – AI Supply Chain Control Tower

## 📌 Project Overview
Today I built an **AI Supply Chain Control Tower**, an interactive browser-based simulation that lets users experience the role of a **Head of Operations** managing real-time supply chain disruptions.

The application is built as a **single self-contained HTML file** using **HTML, CSS, and Vanilla JavaScript** with no frameworks or backend services.

---

# 🚀 Features

- Interactive Operations Control Tower Dashboard
- Live KPI Monitoring
- Real-time Operational Alerts
- Random Supply Chain Disruptions
- Decision-based Gameplay
- Dynamic Score System
- Countdown Timer
- Pause & Resume
- Help / Instructions Modal
- Event Log
- Performance Dashboard
- Replay Functionality
- Fully Responsive UI
- Works Completely Offline

---

# 📊 Dashboard KPIs

- Service Level
- Customer Satisfaction
- Inventory Health
- Transportation Efficiency
- Operating Cost
- Revenue Protected
- Overall Score
- Remaining Time

---

# 🚨 Operational Alerts

The simulator generates multiple supply chain incidents such as:

- Port Congestion
- Supplier Delay
- Truck Breakdown
- Warehouse Stock Issues
- Customs Inspection
- Demand Spike
- Factory Machine Failure
- Weather Disruption
- Wrong Inventory Count
- Damaged Shipment

Each alert includes:

- Priority Level
- Business Impact
- Remaining Time
- Multiple Decision Options
