# Day 25 – AI Shark Tank Simulator

## 🚀 Project Overview

Today I built an **AI Shark Tank Simulator**, a single-page HTML application that allows users to pitch startup ideas to four AI investors, answer investor questions, receive investment decisions, and download a pitch report.

---

## ✨ Features

- 🦈 Four AI Investors
  - Venture Capitalist
  - Founder
  - Customer
  - Angel Investor

- 📝 Startup Pitch Form
  - Startup Name
  - Problem Statement
  - Solution
  - Revenue Model
  - Target Audience
  - Funding Ask

- 💬 Interactive Investor Q&A

- 📊 Startup Scorecard
  - Market Potential
  - Innovation
  - Business Model
  - Execution
  - Investment Worthiness

- 💰 Investment Decision
  - Invest
  - Reject
  - Acquire
  - Come Back Later

- 📈 Suggested Valuation

- 💵 Funding Recommendation

- 📄 PDF Pitch Report

- 🏆 Leaderboard

- 🎉 Confetti Animation

- 🌙 Modern Shark Tank Inspired Dark UI

- 📱 Fully Responsive Design
