# Day 42 – Personal Financial Command Center

## Overview
Built an offline AI-powered financial dashboard using HTML, CSS, and Vanilla JavaScript. The application helps students manage expenses, savings, budgeting, financial health, and future financial planning through an interactive interface.

## Features
- Mission Countdown for NIET enrollment
- Financial Health Score
- Expense Tracker
- Savings & Goal Planner
- What-if Financial Simulator
- Investing 101 (Educational SIP & Compounding)
- Reports Dashboard
- AI Financial Insights
- Tips & Resources
- Local Storage Support
- Responsive Dark UI
- Works Completely Offline

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage

## Key Learnings
- Built a complete offline financial dashboard.
- Implemented budgeting and expense tracking.
- Learned financial health scoring concepts.
- Created interactive simulations using JavaScript.
- Improved responsive UI and dashboard design skills.
